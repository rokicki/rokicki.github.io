These are my new records for maximal determinants, based on pairs of circulant matrices. In all cases N is for the size of a 0/1 matrix. There are three types of matrices represented, given by the second column. Type A is a normal 0/1 recirculant pair. Type B is a -1/+1 recirculant pair; normalize to obtain the 0/1 matrix. Type C is a 0/1 recirculant pair but with a border on the top and left that is about half ones and half zeros (see the HTML subpages). The third column is the new determinant and the fourth column is the ratio of this new determinant to the old record.
